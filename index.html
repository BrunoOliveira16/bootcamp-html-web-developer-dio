A história dos computadores

A história dos computadores vai muito além da idade moderna como conhecemos e inicia na idade antiga com ábaco, porém neste tópico, vamos focar nas gerações de computadores criados e como eles eram construidos

Por volta de 1940 tivemos o que conhecemos como a primeira geração de computadores que eram basicamente máquinas eletromecânicas a válvulas e possuiam um tamanho imenso, com uma capacidade enorme de consumo de enrgia, neste períodos tivemos o Harvard Mark I em 1944, Colossusem 1946 e o ENIAC que também foi criado em 1946.

Por volta de 1950 entramos na segunda geração de computadores que marca a chegada da utilização dos transistores nessas novas máquinas, tivemos como destaque o Univac 1101que foi construido no inicio dos anos 1950.

A terceira geração de computadores teve inicio por volta dos anos de 1965 a 1970, com a chegada dos circuitos integrados com transitors, capacitores e resistores na construção dessas novas máquinas, o primeiro microcomputador a ter suceso nessa época foi o DEC PDP-8 que vendeu cerca de 50 mil unidades em 1965.

Em 1970 inicia-se a quarta geração de computadores e com a utilização de microprocessadores e neste período que empresas como a Apple começam a inovar, como por exemplo o lançamento do primeiro computador com interface gráfica que foi o macintosh e desde então vemos toda a sua evolução nos dias de hoje com notebooks, celulares e tablets.

A quinta geração é conhecida como a geração das IA's, Metaverso, blockchains e criptomoedas, que estarão moldando os proóximos passos do futuro e estão aos poucos se tornando presentes em nosso dia a dia.

Oque são clients?

Uma definição para client é "o dispositivo que exige determinado serviço a um servidor", podemos entender então que o cliente faz as solicitações de serviços a um servidor, que é responsável por atender a estas solicitações.

Oque são servidores?

A definição para servidor é "dispositivo que presta serviço aos computadores conectados a ele através de uma rede", neste contexto podemos entender que o servidor é rsponsável por atender as demandas solicitadas pelo client.

Oque é o HTML?

O HTML é uma abreviação de linguagem de marcação de hypertexto, utilizada na construção de páginas web, estes documentos são interpretados por navegadores como o chrome, firefox, edge e etc. É importante ressaltar que o html não é uma linguagem de programação.

Como é a estrutura básica do html

A estrutura básica de um documento html é constituida das seguintes tags:

  1. html: antes de começar a inserir o conteúdo é necessário colocar a tag html.
  2. head: nesta tag é onde estara contida, informações sobre o documento html, que são fundamentais para o navegador e outros serviços da web, porém não ficam visiveis para os usuários, dentro dessa tag é obrigatória a utilização de outra tag, o title, que é onde colocamos o título da página que aparece no navegador.
  3. body: nesta tag é onde fica o corpo do documento, ou seja todo o conteúdo que fica visivel para o usuário e é onde podemos inserir outras tags para textos, listas, seções e etc.

Textos

Para inserir textos em seu documento HTML existem algumas tags especificas para isso que são:

Tag's H1, H2, H3, H4, H5 e H6: são utilizadas para indicar titulos e subtitulos ao longo do documento e o tamanho da fonte segue a ordem decrescente.

Tag p: é utilizada para compor um paragrafo de texto

Tag b: utilizada para transformar o texto em negrito

Tag i: utilizada para transformar o texto em itálico

Tag hr: utilizada para formar uma linha horizontal e não nescessita de fechamento de tag

Tag br: utilizada para executar uma quebra de linha e não nescessita de fechamento de tag

Oque são listas ordenadas?

As listas ordenadas como o próprio nome diz, são listas que são criadas de forma ordenada, onde são identificadas por numeração "1, 2, 3 e etc"

Para utilizar uma lista ordenada, primeiro precisamos utilizar a tag ol e depois dentro dessa tag utilizar suas tag's filhas que são a tag li.

Oque são listas não ordenadas?

As listas não ordenadas, diferentemente das ordenadas são criadas de forma sem a identificação de numeração

Para utilizar uma lista não ordenada, primeiro precisamos utilizar a tag ul e depois dentro dessa tag utilizar suas tag's filhas que são a tag li.
